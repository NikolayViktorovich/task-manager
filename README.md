# REDDWEB

## Описание проекта

REDDWEB — веб-сайт для планирования задач, разработанный для бизнес-решений с поддержкой интеграции в сложные CRM-системы. Проект направлен на оптимизацию управления задачами, обеспечивая гибкость и удобство для корпоративных пользователей.
